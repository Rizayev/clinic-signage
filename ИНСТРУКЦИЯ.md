# Как пользоваться системой «Цифровые экраны клиники»

Инструкция для администратора. Простыми словами.

## Два адреса
- **Панель управления** (где вы всё настраиваете): http://signage.75.119.137.151.sslip.io
- **Экран / плеер** (открывается на самом телевизоре или мониторе): http://signage.75.119.137.151.sslip.io/player

## Логины

| Роль | Email | Пароль |
| --- | --- | --- |
| Главный администратор | `super@clinic.local` | `password` |

## 1. Вход в панель
Откройте панель управления, введите почту и пароль.

## 2. Подключить телевизор (один раз для каждого экрана)
1. На телевизоре откройте адрес плеера: `.../player`
2. На экране появится поле для кода — введите код привязки. Готовый код для устройства TV-01: **`A7K9-22`**.
3. В панели раздел **Устройства** покажет экран как подключённый. Телевизор готов.

## 3. Загрузить картинки и видео
1. Раздел **Медиатека** → кнопка загрузки.
2. Перетащите файлы (MP4, JPG, PNG). Превью и длительность видео определятся автоматически.

## 4. Собрать плейлист (что крутить на экране)
1. Раздел **Плейлисты** → откройте плейлист (есть готовый «Зал ожидания — утро»).
2. Добавьте в него медиа из медиатеки.
3. Для каждого элемента можно задать, сколько секунд показывать.

## 5. Показать плейлист на экране
1. Раздел **Устройства** → у нужного экрана нажмите **«Назначить плейлист»**.
2. Выберите плейлист → экран сразу начнёт его крутить.

## 6. Бегущая строка (текст внизу экрана)
Раздел **Бегущие строки** → измените текст или включите. Появится на всех экранах за ~0.1 секунды.

## 7. Срочное сообщение (поверх всего)
Раздел **Срочные** → напишите текст → **«Показать срочно»**. Сообщение мгновенно перекроет контент на экранах. Когда больше не нужно — **«Снять»**.

## 8. Кто что может (роли)

| Роль | Может |
| --- | --- |
| Super Admin | всё, включая управление пользователями |
| Admin филиала | контент + экраны / зоны / филиалы |
| Контент-менеджер | только контент (медиа, плейлисты, строки, срочные) |
| Наблюдатель | только смотреть |

---

## Главное
- Любое изменение в панели появляется на экранах **сразу** — перезагружать телевизор не нужно.
- Если интернет пропал, экран продолжит крутить последнее из памяти и покажет значок «Нет связи».
- Один плейлист на нескольких экранах играет синхронно (в ногу).
